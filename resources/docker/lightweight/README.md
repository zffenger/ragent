# 轻量级部署方案

本目录提供内存受限环境下的 Docker Compose 配置，适用于本地开发机或低配服务器。所有容器均设置了内存限制，降低整体资源占用。

与上级目录的默认配置相比：

| 对比项 | 默认版本 (`../`) | 轻量级版本 (本目录) |
|--------|-----------------|---------------------|
| 内存限制 | 无限制，各容器按需占用 | 所有容器均设置 `memory limits` |
| 适用场景 | 生产环境 / 资源充足的开发机 | 本地开发 / 低配服务器 / 体验试用 |

## 注意事项

- 内存限制后，**高并发或大规模数据操作可能触发 OOM**，仅建议开发和体验使用
- 如遇到容器被 kill，可适当调高对应服务的内存限制
- 生产环境请使用上级目录的默认配置
